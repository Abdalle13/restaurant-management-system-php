# Restaurant Management System (PHP & MySQL)

## Project Description

This is a Restaurant Management System developed using PHP and MySQL.
The system allows user registration, login, and role-based access to a dashboard.
All website contents are stored and retrieved from the database.

## Features

- User Registration
- User Login & Logout
- Session and Cookie Authentication
- Automatic Session Expire (5 minutes)
- Admin Dashboard (Control Panel)
- Users Management (CRUD)
  - Add User
  - View Users
  - Edit User
  - Delete User

## How to Run the Project
1. Install XAMPP
2. Copy the project folder to `htdocs`
3. Import `restaurant_db.sql` using phpMyAdmin
4. Run the project in browser:
   http://localhost/restaurant-management-system-php/
